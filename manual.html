this site Manual

About

this site is a PHP/MySQL web application built on the cityphp framework. It's a simple application; you get a list of navigation links and a content page for each link. It's mobile compatible and comes with a CMS.

Installation

First upload the cityphp directory to your web server. cityphp is located here. Next create a MySQL database for the application. Now set these variables in deploy/constants.php:

  • CITY_PHP - an absolute path pointing to the cityphp directory
  • THIS_SITE_PHP - an absolute path pointing to the thissitephp directory
  • ROOT - an absolute path pointing to your web application root. For example, if you are going to host the application at http://mysite.com/myapp/ then you'd use '/myapp/'.
  • DATABASE_HOST - the MySQL database host
  • DATABASE_USERNAME - the MySQL database username
  • DATABASE_PASSWORD - the MySQL database password
  • DATABASE_NAME - the MySQL database name

Disable magic quotes in the cms directory. Upload all the files in the deploy directory to the web application root on your web server. Using a web browser visit install.php in the application root. If the installation was successful, then you'll see a link to the CMS. Delete install.php from your web server and password protect the cms directory.

Usage

Here is a description of each CMS field:

  • Link Title - the navigation link title
  • URL ID - the identifier used in the URL. This can be blank for one link(usually your first link).
  • HTML Title - the title for the HTML title tag
  • HTML Description - the HTML meta description
  • Content - the page content. It's recommended you use only p tags and img tags(see below).
  • Link Order - an integer for the navigation link order
  • Display Mode - display mode of the page. 'Show All' will display the navigation link and the page. 'Hide Link' will hide the navigation link but the page will still display. 'Hide All' will hide the navigation link and the page will not display.

For the CMS content field it is recommended you use only p tags and img tags. For example:

<p>text here..</p>

or:

<p>text here..</p><img src="mypic.jpg"/><p>more text here..</p>
